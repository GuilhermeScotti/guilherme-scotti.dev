# Personal CV Website

This repository contains my CV implemented as a simple HTML website.

The goal is to keep the CV version-controlled, easily maintainable, and accessible online.

The [Live Website](https://guilhermescotti.github.io/guilherme-scotti.dev/) offers a button to generate a PDF version.

## Technologies

- HTML
- CSS
- GitHub Pages
- GitHub Actions (YAML)
